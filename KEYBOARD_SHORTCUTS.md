# Keyboard Shortcuts & UX Guide

Onot is designed for keyboard-first productivity. This guide covers all keyboard shortcuts and UX features.

## Quick Access

Press **Ctrl+?** (or **Cmd+?** on Mac) anytime to view the interactive keyboard shortcuts modal.

---

## General Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl+K` | Open quick search / find pages |
| `Ctrl+P` | Quick page switcher (same as search) |
| `Ctrl+\` | Toggle sidebar visibility |
| `Ctrl+?` | Show keyboard shortcuts help |
| `Ctrl+D` | Toggle dark mode (when not in a block) |
| `Ctrl+N` | Create new page |
| `Esc` | Close modals / Cancel actions |

---

## Block Editing

| Shortcut | Action |
|----------|--------|
| `/` | Open block type menu |
| `Enter` | Create new block below current |
| `Shift+Enter` | New line within current block |
| `Backspace` | Delete empty block / Merge with previous |
| `Ctrl+↑` | Move block up |
| `Ctrl+↓` | Move block down |
| `Ctrl+D` | Duplicate current block |
| `Ctrl+Shift+D` | Delete current block |

## Context Menus (Right-Click)

### Block Context Menu
Right-click on any block to access:
- **Duplicate** - Create a copy of the block
- **Turn into...** - Change block type
- **Color** - Change text/background color
- **Move up/down** - Reorder blocks
- **Delete** - Remove the block

### Page Context Menu (Sidebar)
Right-click on any page in the sidebar to:
- **Add sub-page** - Create a nested page
- **Copy page title** - Copy to clipboard
- **Rename** - Edit page title
- **Delete page** - Remove page and children

## Mouse Interactions

### Block Hover Actions
When you hover over any block, you'll see:
- **Grip icon** (⋮⋮) - Drag to reorder blocks
- **Block type icon** - Click to open action menu

### Page Hover Actions
In the sidebar, hovering over a page shows:
- **+ icon** - Add sub-page
- **⋯ icon** - Open page menu

## Tips & Tricks

1. **Keyboard-First Workflow**: Try to use keyboard shortcuts for everything. It's much faster!

2. **Quick Block Creation**: Type `/` and start typing to filter block types instantly.

3. **Block Navigation**: Use `Ctrl+↑/↓` to quickly reorganize your content.

4. **Right-Click Everything**: Context menus are available on blocks, pages, and more.

5. **Search is Powerful**: Use `Ctrl+K` to quickly jump to any page.

6. **Duplicate for Templates**: Use `Ctrl+D` to duplicate blocks with complex formatting.

7. **Dark Mode Toggle**: Quickly switch themes with `Ctrl+D` (when not editing).

## Platform-Specific Notes

### Windows/Linux
- Use `Ctrl` for all shortcuts
- Shortcuts display as `Ctrl+Key`

### macOS
- Use `⌘` (Command) instead of `Ctrl`
- Shortcuts display as `⌘Key`
- `Alt` is shown as `⌥` (Option)
